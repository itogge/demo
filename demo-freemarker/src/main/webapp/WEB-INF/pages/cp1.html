1、基本取值

  • 整数：${intVar}
  • 长整数：${longVar}
  • 字符串：${stringVar}
  • 双精度：${doubleVar}
  • 日期：${dateVar?string('yyyy-MM-dd HH:mm:ss')}
  • null：${nullVar!'我是默认值'}
  • booleanVar：${booleanVar}
  • missing：${sssVar!}

2、封装类型

  • 对象：${userObject.username}
    对象中的富文本：${(userObject.remark)!?html}

3、集合List

    <#if strList??> <#list strList as item>
  • ${item!}

4、集合Map

    <#list map?keys as key>
  • ${key}:${map[key]}

5、if

  • if
    • <#assign var = 100 /> <#if var == 99> var = 999 <#if var == 99> var = 99 <#else> var != 99 <#if var > 99> var 大于 99 <#elseif var == 99> var = 99

6、switch

  • switch case break default
    • <#assign var = 10 /> <#switch var> <#case 10> 10
    <#break> <#case 11> 11
    <#default> 1223
